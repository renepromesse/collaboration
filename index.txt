technical terms
---------------
- branches: 
    a container that holds some content(files)
how to check branches?
- git branch
how to create a new branch?
- git branch branchName
- ex: git branch tast1
how to change a branch?
- git checkout branchName
- ex: git checkout tast1

how to do get on your local machine the changes from github?
- git pull
- this command will update your local machine with new changes
